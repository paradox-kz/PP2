# 🎵 Music Player with Keyboard Controller

## Objective

Build an interactive music player using **Python** and **Pygame** with keyboard controls.

---

## Features

* Play music
* Stop playback
* Next track
* Previous track
* Display current track information
* Show playback progress bar
* Playlist management
* Keyboard controller support

---

## Keyboard Controls

| Key   | Action         |
| ----- | -------------- |
| **P** | Play           |
| **S** | Stop           |
| **N** | Next track     |
| **B** | Previous track |
| **Q** | Quit           |

## Description

This project is a simple music player application that allows the user to control playback using keyboard commands.

The player supports multiple audio tracks stored in the `music` folder and displays:

* current track name
* playing/stopped status
* playback progress bar
* current time position

---

## Example Usage

Press **P** to start music playback.

Use **N** and **B** to switch between tracks.

Press **S** to stop playback.

Press **Q** to exit the application.
