# 🔴 Moving Ball Game

## Objective

Create an interactive game with a moving red ball using pygame.

---

## Features

* Red ball on white background
* Movement with arrow keys
* 20 pixels per key press
* Boundary checking
* Ball cannot leave screen

---

## Controls

* ↑ Up
* ↓ Down
* ← Left
* → Right

---

## Run

```bash
python main.py
```

---

## Files

* `main.py` → game loop
* `ball.py` → ball logic
* `README.md` → project description
