# 🕒 Mickey's Clock Application

## Objective

Create a digital-style clock using Mickey Mouse hand graphics.

---

## Features

* Real-time system clock
* Minutes and seconds hands
* Mickey hand image rotation
* Updates every second
* Synchronized with system time

---

## Files

* `main.py` → main loop
* `clock.py` → clock logic
* `images/mickey_hand.png` → hand image

---

## Run

```bash id="4d5zj8"
python main.py
```
